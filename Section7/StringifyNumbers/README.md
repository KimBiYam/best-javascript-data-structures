### 개요

객체를 인자로 받아서 객체 안의 모든 숫자 값을 문자열로 변경해서 반환하는 `stringifyNumbers` 함수를 작성한다.

```js
/*
let obj = {
    num: 1,
    test: [],
    data: {
        val: 4,
        info: {
            isRight: true,
            random: 66
        }
    }
}
/*

stringifyNumbers(obj)

/*
{
    num: "1",
    test: [],
    data: {
        val: "4",
        info: {
            isRight: true,
            random: "66"
        }
    }
}
*/
```
